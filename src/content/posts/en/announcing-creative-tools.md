# Announcing New Creative Tools Collection

We're excited to announce the release of our new creative tools collection. This curated set of browser-based tools is designed for designers and developers exploring generative art and interactive experiences.

## What's Included

The collection includes:

- **Particle Designer** - Create and export particle systems
- **Noise Generator** - Generate various types of noise patterns
- **Color Palette Tool** - AI-powered color scheme generation
- **SVG Pattern Maker** - Create seamless vector patterns

## Getting Started

To get started, simply visit the tools page and select the tool you want to use. All tools work directly in your browser with no installation required.

```javascript
// Example: Using the Noise Generator API
const noise = new NoiseGenerator({
  type: 'perlin',
  scale: 0.01,
  octaves: 4
});

const value = noise.get(x, y);
```
